# 📂 配置分类说明

> 以下是插件完整配置的分文件说明，主配置统一由 `_conf_schema.json` 管理，
> 此目录为各类配置的**参考分类说明**，方便查阅和维护。

## 目录结构

```
config/
├── README.md          ← 本文件
├── smtp.json          ← SMTP邮件服务配置
├── verification.json  ← 验证模式与超时配置
├── visual.json        ← 视觉与体验配置
├── templates.json     ← 消息模板配置
└── admin.json         ← 管理与群组配置
```
