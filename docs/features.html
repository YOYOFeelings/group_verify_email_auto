功能介绍

全方位了解 QQ群邮箱验证码插件 的核心能力

双验证体系

两种验证方式，灵活应对不同场景

📧

邮箱验证

系统自动获取用户 QQ 号，拼接为 QQ号@qq.com 发送验证码邮件。用户查看邮箱获取验证码后，在群内 @机器人 回复即可完成验证。

  • 自动获取 QQ 号，无需手动输入邮箱
  • 支持自定义邮件模板和背景图
  • 验证码邮件美观大气
  • 支持多种邮箱 SMTP 配置
➗

数学题验证

系统随机生成加减法数学题，用户回答正确即可通过验证。无需配置邮箱，快速便捷。

  • 随机出题，防自动化破解
  • 无需邮箱配置，开箱即用
  • 验证速度快，用户体验好
  • 适合快速验证场景

插件支持 3 种验证模式：模式 0 让用户自由选择验证方式、模式 1 强制邮箱验证、模式 2 强制数学题验证。管理员可在配置中灵活切换。

核心特性

更多强大功能，全方位守护群聊

📝

自定义模板

所有消息模板均支持变量替换，可自定义入群提示、验证菜单、欢迎语等全部文案。邮件模板支持 5 种预设风格 + 自定义 HTML，满足个性化需求。

图片占位
🔄

回归用户检测

自动识别之前验证成功的用户，重新入群时跳过验证流程。基于 SQLite 数据库记录，数据持久化不丢失。

图片占位
📊

数据库记录

使用 SQLite 持久化存储所有验证记录，包括用户 QQ 号、昵称、邮箱、验证方式、验证结果、时间戳等。支持数据统计和 HTML 报告导出。

图片占位
🌐

多群支持

可指定特定群开启验证，也可配置全局生效。每个群独立管理，互不干扰。支持按群查看统计数据和管理配置。
