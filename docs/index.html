双验证体系

邮箱验证 + 数学题验证，用户可自选或管理员强制指定，灵活应对不同场景。

📧

邮件美化

支持 5 种预设 HTML 模板 + 二次元背景图 + 毛玻璃效果，验证邮件也能赏心悦目。

⏰

超时踢出

验证超时自动踢出，带倒计时警告和最后通牒，防止死群。

防刷保护

邮件发送冷却机制，防止恶意骚扰验证，保障系统稳定运行。

管理指令

测试邮件、查看状态、发送日志、统计数据等完整管理功能，一应俱全。

数据库

SQLite 持久化存储，验证记录永不丢失，支持回归用户自动跳过验证。
