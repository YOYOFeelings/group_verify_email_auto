安装指南

方式一：AstrBot 市场安装

  1. 打开 AstrBot 管理面板
  2. 进入插件市场
  3. 搜索 "群邮箱验证码"
  4. 点击安装即可

方式二：手动安装

  1. 进入 AstrBot 插件目录并克隆仓库：
cd AstrBot/plugins/
git clone https://github.com/YOYOFeelings/group_verify_email_auto.git
  1. 重启 AstrBot
  2. 在管理面板中启用插件

SMTP 配置说明

使用邮件验证功能前，需要先配置 SMTP 邮箱服务。以下是常见邮箱的 SMTP 参数：

邮箱 SMTP主机 端口 加密 密码
QQ邮箱 smtp.qq.com 465 SSL 授权码
163邮箱 smtp.163.com 465 SSL 授权码
Gmail smtp.gmail.com 587 TLS 应用密码
Outlook smtp.office365.com 587 TLS 应用密码

国内邮箱请使用授权码而非登录密码作为 SMTP 密码，需在邮箱设置中开启 SMTP 服务后获取。

管理员设置

安装完成后，在 AstrBot 管理面板的插件配置中设置管理员 QQ 号：

{
  "admin_qqs": ["你的QQ号"]
}

配置 admin_qqs 后，该 QQ 号即可使用管理员指令管理插件。
